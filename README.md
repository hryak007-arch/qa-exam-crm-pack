# QA Premium Pack (Exam 2nd/3rd report add-ons)

Цей пакет містить допоміжні файли, на які посилається звіт:
- `tests/` — інтеграційні тести pytest + FastAPI TestClient
- `scripts/` — seed/cleanup (опис + приклад)
- `reports/` — приклади виходів Bandit/Semgrep (після виправлень)
- `requirements-dev.txt` — dev-залежності для тестів/аналізаторів

> Порада: якщо у вас вже є репозиторій з кодом CRM, просто скопіюйте ці каталоги в корінь проєкту.
